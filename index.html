Membuat bagian header website

1. Untuk memulai membuat website, seperti biasa kita butuh struktur HTML. Agar HTML kita terintegrasi dengan bootstrap kita bisa tambahkan link cdn dan css bootstrap

2. Kita buat container atau pembungkus untuk bagian header yang kita kasih class dengan nama halaman1

3. Kita tambahkan navbar yang bisa kita ambil dari bootstrap lalu kita sesuaikan elemenya dengan header yang ingin kita buat

4. Kita buat lagi container di dalam halaman1 yang berisi satu baris dan dua kolom

5. Untuk kolom pertama kita bisa tambahkan nama, passion, tombol hubungi dan sosial media

6. Untuk kolom kedua kita bisa tambahkan gambar dengan class bootstrap img-fluid untuk responsif
